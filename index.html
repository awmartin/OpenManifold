OpenManifold by awmartin

Software Platform for Performative Architectural Design

OpenManifold is built to enable an architect to work on a "higher level" of system-based relationships, topologies, and simulation environments. The geometry engine is intended to be robust enough to handle the details of creating parametric, geometric objects nimbly and efficiently. There are then analysis and algorithmic engines which resolve the given relationships according to some notion of performative optimization.

Dependencies

OpenManifold has no dependencies.

Install

Download the package, OpenManifold.dmg, open, and copy the application OpenManifold.app into the Applications folder.
