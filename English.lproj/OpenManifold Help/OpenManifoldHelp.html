OpenManifold Help

OpenManifold is a parametric and algorithmic modeler. Based on the best techniques of existing software like Generative Components, Grasshopper, Rhino, Inventor, Revit, and Processing, this software intends to take a higher-level approach to generative design by allowing the designer to focus on high-level behaviors and relationships, instead of the nitty-gritty details of geometry.

The current project help can be found at http://labs.anomalus.com/

Ruby scripting commands

@openmanifold
This object controls the document.
@openmanifold.addPart
Adds a new empty part to the document. These parts contain geometries (surfaces, points, curves, etc.) that are parametrically related. You can specify as many parameters as you like.

Example: Adding an part

@part = @openmanifold.addPart
That's it. Type this into the script window and click 'Evaluate' or press cmd+return. A wireframe cube should appear in the field.
